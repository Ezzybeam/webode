Install

1
Tampermonkey — easiest, no dev mode needed
Install Tampermonkey from the Chrome Web Store, then click the button below.
Install script
2
Chrome Extension — requires Developer Mode
Download the repo, go to chrome://extensions, enable Developer Mode, click Load Unpacked, select the webode-extension folder.
Download zip
3
DevTools Overrides — manual
Open DevTools → Sources → Overrides → Enable Local Overrides, then save and replace the game files from the www.geometrydash.com/game/ folder in the repo.
